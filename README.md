# EggTimer

iOS app to prepare boiled eggs. It gives an alarm when the cooking time is over. Choose between soft, medium or hard.

Concepts:

* Swift Collection types - Dictionaries
* The Swift Timer API
* Functions with outputs
* How to use the ProgressView
